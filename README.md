<H1> 🚀 Gerenciador de Projetos Java </H1>
Este projeto foi desenvolvido como parte do currículo acadêmico da faculdade, com o objetivo de criar uma ferramenta para a gestão de projetos. A aplicação permite o controle total do ciclo de vida de um projeto, desde a sua concepção até a entrega final, utilizando metodologias visuais como o Kanban.
